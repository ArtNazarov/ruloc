# RULOC - Мониторинг и блокировка соединений не из России

Скрипт `ruloc.py` отслеживает установленные TCP-соединения на сервере и автоматически реагирует на соединения из стран, отличных от России. Он может блокировать IP-адреса через `ufw` и/или завершать процессы, инициировавшие нежелательные соединения.

## Возможности

- Мониторинг активных TCP-соединений в реальном времени
- Определение страны по IP-адресу (используется база GeoLite2)
- Автоматическая блокировка IP через `ufw`
- Завершение процессов с нежелательными соединениями
- Только соединения из `RU` считаются разрешёнными

## Требования

- Python 3.6+
- Linux с `ufw` (для блокировки)
- Права `sudo` (для добавления правил в `ufw`)
- Файл базы данных `GeoLite2-Country.mmdb` в директории со скриптом

## Установка и настройка

### 1. Клонирование репозитория

```bash
git clone <your-repo-url>
cd <project-directory>
```

### 2. Настройка виртуального окружения

```bash
# Создание виртуального окружения
python3 -m venv venv

# Активация виртуального окружения
source venv/bin/activate   # Для Linux/macOS
# или
venv\Scripts\activate      # Для Windows (если используется локально)
```

### 3. Установка зависимостей

```bash
pip install psutil geoip2
```

### 4. Загрузка базы данных GeoIP

Скачайте бесплатную базу `GeoLite2-Country.mmdb` с [официального сайта MaxMind](https://dev.maxmind.com/geoip/geolite2-free-geolocation-data) и поместите её в директорию со скриптом.

```bash
wget https://github.com/P3TERX/GeoLite2.mmdb/raw/master/GeoLite2-Country.mmdb
```

### 5. Настройка прав sudo

Для работы блокировки через `ufw` скрипт требует прав `sudo` без запроса пароля (рекомендуется только для конкретной команды):

```bash
sudo visudo -f /etc/sudoers.d/ufw-ruloc
```

Добавьте строчку:
```
your_username ALL=(ALL) NOPASSWD: /usr/sbin/ufw deny from *
```

## Использование

### Базовый синтаксис

```bash
python3 ruloc.py [--ufw-block-detected] [--close-proc]
```

### Примеры запуска

#### Только блокировка IP через UFW

```bash
python3 ruloc.py --ufw-block-detected
```

#### Только завершение процессов

```bash
python3 ruloc.py --close-proc
```

#### Блокировка + завершение процессов

```bash
python3 ruloc.py --ufw-block-detected --close-proc
```

### Запуск в фоновом режиме

```bash
nohup python3 ruloc.py --ufw-block-detected --close-proc &
```

Или с использованием `screen`/`tmux`:

```bash
screen -S ruloc
python3 ruloc.py --ufw-block-detected --close-proc
# Ctrl+A, D - отсоединиться
```

## Как это работает

1. Скрипт каждые 3 секунды сканирует все установленные TCP-соединения
2. Для каждого внешнего IP определяется страна по базе GeoIP
3. Если страна не `RU`:
   - Выводится сообщение в консоль
   - При `--ufw-block-detected` добавляется правило `ufw deny from <IP>`
   - При `--close-proc` завершается процесс, владеющий соединением

## Важно

- Скрипт не отличает входящие и исходящие соединения — проверяются **все** установленные соединения
- Блокировка через `ufw` перманентна до ручного удаления правила
- Завершение процессов может быть опасным — используйте опцию `--close-proc` с осторожностью на production-серверах
- Для корректной работы необходима актуальная база GeoIP

## Просмотр добавленных правил UFW

```bash
sudo ufw status numbered
```

## Удаление правил UFW

```bash
sudo ufw delete <номер_правила>
```

## Остановка скрипта

Нажмите `Ctrl+C` в терминале или завершите процесс:

```bash
pkill -f ruloc.py
```

## Удаление виртуального окрущения

```bash
deactivate
rm -rf venv
```
